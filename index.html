Five minute starter path

Follow these moves in a fresh folder. When you are done, you will have your first commit pushed.

  1. Initialize: run git init and add a README.
  2. Stage & commit: git add . then git commit -m "first commit".
  3. Create remote: make a repo on GitHub and copy its URL.
  4. Connect: git remote add origin <url>.
  5. Push: git branch -M main then git push -u origin main.
Tips as you go
  • Commit often; keep messages short and clear.
  • Use git status whenever you're unsure.
  • Practice branching: git checkout -b feature/idea.
  • Break things safely with git switch and git stash.
  • Read history with git log --oneline --graph.
FAQ

What do I install? Git CLI and a GitHub account. Optional: VS Code + GitLens.

How do I undo? Use git restore for files, git revert for commits, and git reset --soft when you need to redo staging.

What next? Try a branch-only workflow: open a PR, request a review, and merge via GitHub.
